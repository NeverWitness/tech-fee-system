# 文档结构

本项目的文档分为以下几个部分。

---

## 一、问题背景

```
docs/background/
```

这一部分用于说明问题的现实背景，包括：

- 为什么资产差距难以消除
- 为什么生活差距比资产差距更重要

该部分构成整个讨论的起点。

---

## 二、价格结构

```
docs/price-structure/
```

这一部分讨论现代经济中的价格结构问题，包括：

- 基础生活品的边界
- 为什么实现生活保障需要重构物价结构

这一部分为后续讨论提供经济结构基础。

---

## 三、技术费用框架

```
docs/technology-fee/
```

这是项目的核心部分，主要讨论：

- 技术费用的基本概念
- 技术层级与知识扩散
- 技术费用的动态变化
- 技术公开与授权使用
- 技术如何通过人和组织转化为企业竞争力

---

## 四、产业结构

```
docs/industry/
```

这一部分通过概述性分析说明：

不同产业中的技术费用比例差异，例如：

- 农业
- 基础工业
- 传统服务业
- 数字产业

正文只提供整体结构概述。

---

## 五、制度讨论

```
docs/policy-discussion/
```

这一部分讨论技术时代可能出现的一些制度问题，包括：

- 动态全民补助机制
- 传统税收体系在技术时代面临的困境
- 技术费用如何回应技术时代的税收问题
- 价格压缩是否会导致供给侧扭曲
- 技术费用制度可能面临的外部挑战

---

## 六、制度演化

```
docs/evolution/
```

这一部分讨论制度在现实中的可能发展路径，例如：

- 企业策略与技术分工
- 制度演化路径

---

## 七、附录

```
docs/appendix/
```

附录包含若干行业案例，用于说明技术费用在不同产业中的具体表现，例如：

- 农业产品
- 高端农业产品
- 基础工业产品
- 高端工业产品
- 传统服务业
- 2C 数字服务
- 2B 数字服务

附录中的案例主要用于说明概念，不构成完整产业研究。
